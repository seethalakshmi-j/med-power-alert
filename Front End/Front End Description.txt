Front-End Description: 

The Hospital Voltage Monitoring System front end is a simple, intuitive, and responsive web interface designed to visually monitor electrical voltage levels in critical hospital environments. The interface allows users to manually input voltage values and instantly receive real-time status feedback through color-coded alerts.

The dashboard is built using HTML, CSS, and JavaScript, ensuring fast performance and ease of use even for non-technical staff. A minimal and centered layout improves readability and focuses user attention on critical alerts.
